Angular-capitalize

A simple way to capitalize you strings with angularJS

View on GitHub Download .zip Download .tar.gz

Welcome to angular-capitalize !

angular-capitalize is an angular's filter that allow you to capitalize your strings

Use the filter just for the first word : | capitalize

{{capitalizeCtrl.someString | capitalize }}

Use the filter for all the words : | capitalize:true

{{capitalizeCtrl.someString | capitalize:true }}
